## 欢迎

在 Mintlify 中有两种构建 API 文档的方式：
- 基于 OpenAPI 规范的自动生成
- 基于 MDX 组件的手动编写

本项目的示例使用一份 OpenAPI 规范文件，你可以在下方查看。

## 认证方式

所有 API 端点均使用 Bearer Token 认证，示例片段：

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

在请求头添加 `Authorization: Bearer <token>` 即可。
