## 提交任务

- 请求头: `Authorization: Bearer <your_api_key>`
- 必需: `text`（1-5000 字符）
- 可选: `voice`, `stability`, `similarity_boost`, `style`, `speed`, `timestamps`, `previous_text`, `next_text`, `language_code`
- 返回 `task_id`；通过 `/api/v1/task/status` 查询状态
- 比 TTS V2 **便宜 50%**，针对低延迟优化

### 基础示例

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/eleven-labs-tts-turbo-25 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "欢迎使用我们的人工智能平台。"
  }'
```

### 完整参数示例

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/eleven-labs-tts-turbo-25 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "解锁强大的 API 平台。",
    "voice": "Brian",
    "stability": 0.5,
    "speed": 1.0
  }'
```

<Note>
- **文本**: 1-5000 字符，支持多语言。
- **语音**: 21 种选择（Rachel, Brian, Sarah 等）。默认: Rachel。
- **语速**: 0.7-1.2（默认: 1.0）。
- **价格**: 每 1000 字符 40 积分（比 V2 便宜 50%）。
</Note>
