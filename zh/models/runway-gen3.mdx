## 快速上手

- 平台：Wizzx（第三方 API 聚合），统一使用 `Authorization: Bearer <your_api_key>`。
- 提交任务：`POST https://api.wizzx.ai/api/v1/task/submit/runway-gen3`，异步返回 `task_id`。
- 轮询状态：`POST https://api.wizzx.ai/api/v1/task/status`，传入 `task_id` 获取结果。
- 模式：`text-to-video`（纯文本）、`image-to-video`（1 张参考图）。

### 认证

```bash
Authorization: Bearer <your_api_key>
```

### 提交任务

接口：`POST https://api.wizzx.ai/api/v1/task/submit/runway-gen3`，`Content-Type: application/json`

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `prompt` | string | 是 | - | 描述视频内容的文本提示词 |
| `type` | string | 是 | - | `text-to-video` 或 `image-to-video` |
| `image_urls` | array | 条件必填 | - | 图片 URL 数组（`image-to-video` 必填，最多 1 张） |
| `duration` | int | 否 | 5 | 视频时长：`5` 或 `10` 秒 |
| `resolution` | string | 否 | `720p` | `720p` 或 `1080p`（10 秒不支持 1080p） |
| `aspect_ratio` | string | 否 | auto | `16:9`、`9:16` 或 `1:1` |

<Warning>
  **限制：** 10 秒视频不支持 1080p 分辨率。
</Warning>

#### 请求示例：文生视频

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/runway-gen3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "电影级的无人机穿越霓虹城市夜景",
    "type": "text-to-video",
    "duration": 5,
    "resolution": "720p",
    "aspect_ratio": "16:9"
  }'
```

#### 请求示例：图生视频

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/runway-gen3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "让角色优雅地向前走动",
    "type": "image-to-video",
    "image_urls": ["https://example.com/character.jpg"],
    "duration": 5,
    "resolution": "720p"
  }'
```

成功响应：

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "task_id": "task_123456789",
    "status": "pending",
    "credits_deducted": 5
  }
}
```

### 查询状态

接口：`POST https://api.wizzx.ai/api/v1/task/status`

```json
{
  "task_id": "task_123456789"
}
```

响应示例（成功）：

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "task_id": "task_123456789",
    "status": "succeeded",
    "result_urls": ["https://cdn.example.com/video.mp4"]
  }
}
```

状态流程：`pending`（排队中）→ `processing`（生成中）→ `succeeded`（成功，返回 `result_urls`）或 `failed`（失败，查看 `error`）。

### 计费

| 时长 | 分辨率 | 积分 |
|------|--------|------|
| 5 秒 | 720p | 5 |
| 5 秒 | 1080p | 10 |
| 10 秒 | 720p | 5 |
| 10 秒 | 1080p | 不支持 |

### 最佳实践

1. **时长与分辨率：**
   - 较长视频（10 秒）请使用 720p 分辨率
   - 1080p 仅支持 5 秒视频

2. **模式选择：**
   - 使用 `text-to-video` 进行纯文本驱动生成
   - 使用 `image-to-video` 时提供参考图可获得更好的一致性

3. **宽高比：**
   - 横屏视频使用 `16:9`
   - 竖屏视频使用 `9:16`
   - 方形视频使用 `1:1`
   - 图生视频建议留空由系统自动检测
