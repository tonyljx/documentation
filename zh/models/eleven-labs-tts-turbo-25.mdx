## 概述

- **提供商**: ElevenLabs
- 通过 `POST https://api.wizzx.ai/api/v1/task/submit/eleven-labs-tts-turbo-25` 提交任务；使用 `task_id` 在 `POST https://api.wizzx.ai/api/v1/task/status` 查询状态。

## 查询状态

- 端点: `POST https://api.wizzx.ai/api/v1/task/status`
- 请求体:

```json
{
  "task_id": "task_123456789"
}
```

### 状态响应

```json
{
  "code": 0,
  "data": {
    "task_id": "task_123456789",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.wizzx.ai/audio/output.mp3"
    ],
    "error": ""
  }
}
```

状态: `PROCESSING`（生成中）, `SUCCEEDED`（成功，使用 `result_urls`）, `FAILED`（失败，`error` 包含错误信息）。

## 常见问题

- **何时使用 Turbo 2.5 vs V2？** Turbo 2.5 适合实时应用、聊天机器人或对成本敏感的场景。V2 适合需要最高品质的高端内容。
- **质量差异明显吗？** Turbo 2.5 提供适合大多数场景的高质量输出。V2 为发烧级需求提供略高的质量。
- **可以在两个模型间切换吗？** 可以，两个模型使用相同的参数。只需更改端点即可。
