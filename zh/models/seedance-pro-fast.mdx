## 快速上手

- **重大变更（2025-12-10）**：`mode` → `type`；`input_image_url` → `image_urls`（数组）。
- 品牌：Wizzx（第三方 API 聚合），统一使用 `Authorization: Bearer <your_api_key>`.
- 提交生成：`POST /api/v1/task/submit/seedance-pro-fast`，异步返回 `task_id`。
- 轮询状态：`POST /api/v1/task/status`，传入 `task_id` 获取结果或错误。
- 支持 `text-to-video` 与 `image-to-video` 两种模式；固定 24fps，时长 5s/10s，分辨率 480p/720p/1080p。

### 认证

所有请求需携带：

```bash
Authorization: Bearer <your_api_key>
```

Wizzx 网关会校验并注入身份，无需额外参数。

### 提交任务

接口：`POST /api/v1/task/submit/seedance-pro-fast`，`Content-Type: application/json`

### 必填参数

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `prompt` | string | 视频描述 |
| `type` | string | 生成类型：`text-to-video` 或 `image-to-video` |

### 可选参数

| 字段 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `image_urls` | []string | - | `image-to-video` 必填，最多 1 张 |
| `duration` | int | 5 | 视频时长：`5` 或 `10` 秒 |
| `resolution` | string | 480p | 视频分辨率：`480p` / `720p` / `1080p` |
| `aspect_ratio` | string | 16:9 / adaptive | text-to-video 默认 `16:9`，image-to-video 默认 `adaptive` |
| `seed` | int | -1 | 随机种子，-1 表示随机，范围 [-1, 4294967295] |
| `watermark` | bool | false | 是否添加水印 |
| `camera_fixed` | bool | false | 是否固定镜头 |

<Warning>
  **图片要求**（img2video 模式）：
  - 格式：jpeg/png/webp/bmp/tiff/gif
  - 宽高比：0.4-2.5
  - 短边 `>300px`，长边 `<6000px`
  - 大小 `<30MB`
</Warning>

#### 请求示例（Text2Video）

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A cat playing piano in a cozy living room",
    "type": "text-to-video",
    "duration": 5,
    "resolution": "480p",
    "aspect_ratio": "16:9"
  }'
```

#### 请求示例（Image2Video）

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Make the character walk forward gracefully",
    "type": "image-to-video",
    "image_urls": ["https://example.com/character.jpg"],
    "duration": 10,
    "resolution": "720p",
    "aspect_ratio": "adaptive"
  }'
```

成功响应示例：

```json
{
  "code": 0,
  "data": {
    "task_id": "task_123456789",
    "model_key": "seedance-pro-fast"
  }
}
```

### 任务查询

接口：`POST /api/v1/task/status`

请求体：

```json
{
  "task_id": "task_123456789"
}
```

响应示例：

```json
{
  "code": 0,
  "data": {
    "task_id": "task_123456789",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.example.com/video.mp4"
    ],
    "error": ""
  }
}
```

状态值：`PROCESSING`（生成中）、`SUCCEEDED`（可用 `result_urls`）、`FAILED`（查看 `error`）。

### 常见问题

- **`mode` 字段去哪了？** 统一改为 `type`，值为 `text-to-video` / `image-to-video`。
- **如何控制生成一致性？** 传 `seed`，相同 seed + 参数会生成相似结果。
- **image-to-video 下宽高比如何处理？** 默认 `adaptive` 自动选择最佳比例，手动指定时系统会按比例裁剪。
- **fps 能否调整？** Seedance Pro Fast 固定 24fps。
- **camera_fixed 有什么用？** 开启后镜头固定，更适合稳定画面。
