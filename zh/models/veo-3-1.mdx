## 快速上手

- **重大变更（2025-12-10）**：`generation_type` → `type`；枚举改为 `text-to-video` / `image-to-video` / `reference-to-video`。
- 品牌：Wizzx（第三方 API 聚合），统一使用 `Authorization: Bearer <your_api_key>`.
- 提交生成：`POST /api/v1/task/submit/veo3`，异步返回 `task_id`。
- 轮询状态：`POST /api/v1/task/status`，传入 `task_id` 获取结果或错误。
- 默认模型 `veo3_fast`，16:9 画幅；支持文本驱动、首帧/首尾帧和参考图模式。

### 认证

所有请求需携带：

```bash
Authorization: Bearer <your_api_key>
```

Wizzx 网关会校验并注入身份，无需额外参数。

### 提交任务

接口：`POST /api/v1/task/submit/veo3`，`Content-Type: application/json`

### 必填参数

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `prompt` | string | 视频描述 |

### 可选参数

| 字段 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `type` | string | 自动推断 | `text-to-video` / `image-to-video` / `reference-to-video` |
| `image_urls` | []string | - | 1-3 张图片（规则见下） |
| `model` | string | veo3_fast | `veo3` / `veo3_fast` / `veo3.1_fast` |
| `aspect_ratio` | string | 16:9 | `16:9` / `9:16` / `Auto` |
| `seeds` | int | - | 10000-99999，可复现 |
| `enable_translation` | bool | true | 是否自动翻译提示词 |
| `watermark` | string | - | 自定义水印文字 |

### type 字段规则

| 类型 | 说明 | image_urls 要求 |
|------|------|-----------------|
| `text-to-video` | 纯文本生成视频 | 不传 `image_urls` |
| `image-to-video` | 首帧/首尾帧生成 | 1-2 张图（1 张=首帧，2 张=首尾帧） |
| `reference-to-video` | 参考图生成 | 1-3 张参考图，需 16:9 + `model=veo3_fast` |

自动推断规则：
- 不传 `image_urls`：默认 `text-to-video`
- 传 1-2 张 `image_urls`：默认 `image-to-video`
- 传 1-3 张且显式 `type="reference-to-video"`：参考图模式

<Warning>
  `reference-to-video` 必须使用 `veo3_fast` 且 `aspect_ratio` 固定 16:9。
</Warning>

#### 请求示例（Text2Video）

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A dog running through a neon city at night",
    "type": "text-to-video",
    "aspect_ratio": "16:9",
    "model": "veo3_fast",
    "seeds": 12345
  }'
```

#### 请求示例（Image2Video，首尾帧）

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Make the scene dynamic while keeping the character consistent",
    "type": "image-to-video",
    "image_urls": [
      "https://example.com/frame1.jpg",
      "https://example.com/frame2.jpg"
    ],
    "aspect_ratio": "9:16"
  }'
```

#### 请求示例（Reference2Video）

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Product turntable style, soft studio light",
    "type": "reference-to-video",
    "image_urls": [
      "https://example.com/ref1.jpg",
      "https://example.com/ref2.jpg",
      "https://example.com/ref3.jpg"
    ],
    "model": "veo3_fast",
    "aspect_ratio": "16:9",
    "enable_translation": true
  }'
```

成功响应示例：

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "provider_task_id": "veo_task_abcdef123456",
    "model_key": "veo3",

```

### 任务查询

接口：`POST /api/v1/task/status`

请求体：

```json
{
  "task_id": "local-task-id"
}
```

响应示例：

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.example.com/video.mp4"
    ],
    "error": ""
  }
}
```

状态值：`PROCESSING`（生成中）、`SUCCEEDED`（可用 `result_urls`）、`FAILED`（查看 `error`）。

### 常见问题

- **`generation_type` 字段去哪了？** 统一改为 `type`，枚举改为小写连字符格式。
- **需要回调 URL 吗？** 不需要，直接轮询 `task/status`。
- **如何复现结果？** 传 `seeds`（10000-99999）。
- **非英文提示词会怎样？** 默认 `enable_translation=true` 自动翻译，可关闭。
- **image-to-video 下 1 张 vs 2 张图？** 1 张作为首帧参考，2 张为首帧+尾帧，系统生成过渡。
