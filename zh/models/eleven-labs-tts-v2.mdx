## 概述

- **提供商**: ElevenLabs
- 通过 `POST https://api.wizzx.ai/api/v1/task/submit/eleven-labs-tts-v2` 提交任务；使用 `task_id` 在 `POST https://api.wizzx.ai/api/v1/task/status` 查询状态。
- 必需: `text`（1-5000 字符）。
- 可选: `voice`, `stability`, `similarity_boost`, `style`, `speed`, `timestamps`, `previous_text`, `next_text`, `language_code`。
- 支持多语言，自动检测语言。

## 认证

- 请求头: `Authorization: Bearer <your_api_key>`

## 提交任务

端点: `POST https://api.wizzx.ai/api/v1/task/submit/eleven-labs-tts-v2`, `Content-Type: application/json`

### 必需参数

| 字段 | 类型 | 说明 |
|------|------|------|
| `text` | string | 要转换的文本（1-5000 字符） |

| `voice` | string | `Rachel` | 语音选择（见下方语音列表） |

## 查询状态

- 端点: `POST https://api.wizzx.ai/api/v1/task/status`
- 请求体:

```json
{
  "task_id": "task_123456789"
}
```

### 状态响应

```json
{
  "code": 0,
  "data": {
    "task_id": "task_123456789",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.wizzx.ai/audio/output.mp3"
    ],
    "error": ""
  }
}
```

状态: `PROCESSING`（生成中）, `SUCCEEDED`（成功，使用 `result_urls`）, `FAILED`（失败，`error` 包含错误信息）。

## 常见问题

- **支持哪些语言？** ElevenLabs 支持多种语言，包括英语、中文、日语、韩语、西班牙语等。模型会自动检测语言。
- **如何选择合适的语音？** 根据内容风格选择 - 专业内容用商务语音，友好内容用温暖语音。
- **生成的音频会保留多久？** 30 天；请及时下载和存档重要文件。
- **如何提高语音质量？**
  - 调整 `stability`：较高值（0.7-1.0）更稳定一致，较低值（0.3-0.5）更富表现力
  - 使用 `previous_text` 和 `next_text` 提高句子之间的连贯性
- **speed 参数如何影响输出？**
  - 0.7-0.9：语速较慢，适合教程或需要清晰表达的内容
  - 1.0：正常语速
  - 1.1-1.2：语速较快，适合快节奏内容
