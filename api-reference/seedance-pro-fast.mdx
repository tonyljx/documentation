## Quick start

- Platform: Wizzx gateway; use `Authorization: Bearer <your_api_key>`.
- Submit: `POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast` (async `task_id`).
- Poll: `POST https://api.wizzx.ai/api/v1/task/status` with `task_id`.
- Modes: `text2video`, `img2video`.
- Fixed 24fps; duration 5s or 10s; resolutions 480p/720p/1080p.

### Auth

```bash
Authorization: Bearer <your_api_key>
```

### Submit job

Endpoint: `POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast`, `Content-Type: application/json`

| Field | Type | Description | Default/Rules |
| --- | --- | --- | --- |
| `prompt` | string | Video description | **Required** |
| `mode` | string | `text2video` or `img2video` | **Required** |
| `input_image_url` | string | Image URL | Required for `img2video` |
| `duration` | int | Seconds | `5` or `10`, default `5` |
| `resolution` | string | `480p` / `720p` / `1080p` | Default `480p` |
| `aspect_ratio` | string | `16:9` / `4:3` / `1:1` / `3:4` / `9:16` / `21:9` / `adaptive` | text2video default `16:9`; img2video default `adaptive` |
| `seed` | int | RNG seed | `-1` random, range [-1, 4294967295] |
| `watermark` | bool | Add watermark | Default `false` |
| `camera_fixed` | bool | Lock camera | Default `false` |

<Warning>
  **Image requirements (img2video)**  
  - Formats: jpeg/png/webp/bmp/tiff/gif  
  - Aspect ratio: 0.4-2.5  
  - Short side \>300px, long side \<6000px  
  - Size \<30MB
</Warning>

#### Example: Text2Video

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A cat playing piano in a cozy living room",
    "mode": "text2video",
    "duration": 5,
    "resolution": "480p",
    "aspect_ratio": "16:9"
  }'
```

#### Example: Image2Video

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Make the character walk forward gracefully",
    "mode": "img2video",
    "input_image_url": "https://example.com/character.jpg",
    "duration": 10,
    "resolution": "720p",
    "aspect_ratio": "adaptive"
  }'
```

Success response:

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "provider_task_id": "seedance_task_abcdef123456",
    "model_key": "seedance-pro-fast",
    "provider_key": "seedance-pro-fast"
  }
}
```

### Query status

Endpoint: `POST https://api.wizzx.ai/api/v1/task/status`

```json
{
  "task_id": "local-task-id"
}
```

Response example:

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.example.com/video.mp4"
    ],
    "error": ""
  }
}
```

Status values: `PROCESSING`, `SUCCEEDED`, `FAILED`.
