## Quick start

- **Breaking change (2025-12-10)**: `mode` → `type`; `input_image_url` → `image_urls` (array).
- Platform: Wizzx gateway; `Authorization: Bearer <your_api_key>`.
- Submit: `POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast` → returns `task_id`.
- Poll: `POST https://api.wizzx.ai/api/v1/task/status` with `task_id`.
- Modes: `text-to-video` / `image-to-video`; fixed 24fps; duration 5s or 10s; resolutions 480p/720p/1080p.

### Auth

```bash
Authorization: Bearer <your_api_key>
```

### Submit job

Endpoint: `POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast`, `Content-Type: application/json`

### Required

| Field | Type | Notes |
| --- | --- | --- |
| `prompt` | string | Video description |
| `type` | string | `text-to-video` or `image-to-video` |

### Optional

| Field | Type | Default | Notes |
| --- | --- | --- | --- |
| `image_urls` | []string | - | Required for `image-to-video`, max 1 image |
| `duration` | int | 5 | `5` or `10` seconds |
| `resolution` | string | 480p | `480p` / `720p` / `1080p` |
| `aspect_ratio` | string | 16:9 / adaptive | text-to-video default `16:9`, image-to-video default `adaptive` |
| `seed` | int | -1 | RNG seed; -1 for random, range [-1, 4294967295] |
| `watermark` | bool | false | Add watermark |
| `camera_fixed` | bool | false | Lock camera position |

<Warning>
  **Image requirements (img2video)**  
  - Formats: jpeg/png/webp/bmp/tiff/gif  
  - Aspect ratio: 0.4-2.5  
  - Short side \>300px, long side \<6000px  
  - Size \<30MB
</Warning>

#### Example: Text2Video

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A cat playing piano in a cozy living room",
    "type": "text-to-video",
    "duration": 5,
    "resolution": "480p",
    "aspect_ratio": "16:9"
  }'
```

#### Example: Image2Video

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Make the character walk forward gracefully",
    "type": "image-to-video",
    "image_urls": ["https://example.com/character.jpg"],
    "duration": 10,
    "resolution": "720p",
    "aspect_ratio": "adaptive"
  }'
```

Success response:

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "provider_task_id": "seedance_task_abcdef123456",
    "model_key": "seedance-pro-fast",
    "provider_key": "seedance-pro-fast"
  }
}
```

### Query status

Endpoint: `POST https://api.wizzx.ai/api/v1/task/status`

```json
{
  "task_id": "local-task-id"
}
```

Response example:

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.example.com/video.mp4"
    ],
    "error": ""
  }
}
```

Status values: `PROCESSING`, `SUCCEEDED`, `FAILED`.

### FAQ

- Where did `mode` go? Unified as `type`, values now `text-to-video` / `image-to-video`.
- How to keep results repeatable? Use `seed`; same seed + params yields similar results.
- How is aspect ratio handled for `image-to-video`? Default `adaptive` auto-selects based on input; manual ratios are cropped automatically.
- Can fps be changed? Fixed at 24fps.
- What does `camera_fixed` do? Locks camera position for a steadier shot.
