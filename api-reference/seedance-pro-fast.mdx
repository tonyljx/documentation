## 快速上手

- 品牌：Wizzx（第三方 API 聚合），统一使用 `Authorization: Bearer <your_api_key>`.
- 提交生成：`POST /api/v1/task/submit/seedance-pro-fast`，异步返回 `task_id`。
- 轮询状态：`POST /api/v1/task/status`，传入 `task_id` 获取结果或错误。
- 支持 `text2video` 和 `img2video` 两种模式。
- 固定 24fps，支持 5 秒或 10 秒时长，480p/720p/1080p 分辨率。

### 认证

所有请求需携带：

```bash
Authorization: Bearer <your_api_key>
```

Wizzx 网关会校验并注入身份，无需额外参数。

### 提交任务

接口：`POST /api/v1/task/submit/seedance-pro-fast`，`Content-Type: application/json`

| 字段 | 类型 | 说明 | 默认/规则 |
| --- | --- | --- | --- |
| `prompt` | string | 视频描述 | **必填** |
| `mode` | string | 生成模式 | **必填**，`text2video` 或 `img2video` |
| `input_image_url` | string | 输入图片 URL | `img2video` 模式**必填** |
| `duration` | int | 视频时长（秒） | `5` 或 `10`，默认 `5` |
| `resolution` | string | 视频分辨率 | `480p` / `720p` / `1080p`，默认 `480p` |
| `aspect_ratio` | string | 宽高比 | `16:9` / `4:3` / `1:1` / `3:4` / `9:16` / `21:9` / `adaptive`，text2video 默认 `16:9`，img2video 默认 `adaptive` |
| `seed` | int | 随机种子 | `-1` 表示随机，范围 [-1, 4294967295] |
| `watermark` | bool | 是否添加水印 | 默认 `false` |
| `camera_fixed` | bool | 是否固定镜头 | 默认 `false` |

<Warning>
  **图片要求**（img2video 模式）：
  - 格式：jpeg/png/webp/bmp/tiff/gif
  - 宽高比：0.4-2.5
  - 短边 `>300px`，长边 `<6000px`
  - 大小 `<30MB`
</Warning>

#### 请求示例（Text2Video）

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A cat playing piano in a cozy living room",
    "mode": "text2video",
    "duration": 5,
    "resolution": "480p",
    "aspect_ratio": "16:9"
  }'
```

#### 请求示例（Image2Video）

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Make the character walk forward gracefully",
    "mode": "img2video",
    "input_image_url": "https://example.com/character.jpg",
    "duration": 10,
    "resolution": "720p",
    "aspect_ratio": "adaptive"
  }'
```

成功响应示例：

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "provider_task_id": "seedance_task_abcdef123456",
    "model_key": "seedance-pro-fast",
    "provider_key": "seedance-pro-fast"
  }
}
```

### 任务查询

接口：`POST /api/v1/task/status`

请求体：

```json
{
  "task_id": "local-task-id"
}
```

响应示例：

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.example.com/video.mp4"
    ],
    "error": ""
  }
}
```

状态值：`PROCESSING`（生成中）、`SUCCEEDED`（可用 `result_urls`）、`FAILED`（查看 `error`）。

### 参数速览与规则

- **必填**：`prompt`、`mode`
- **条件必填**：`input_image_url`（img2video 模式）
- **可选**：`duration`、`resolution`、`aspect_ratio`、`seed`、`watermark`、`camera_fixed`
- **规则提示**：
  - `text2video` 模式不需要 `input_image_url`
  - `img2video` 模式必须提供 `input_image_url`
  - `adaptive` 宽高比会根据输入图片自动选择最佳比例

### 常见问题

- **需要回调吗？** 不需要，使用 `task/status` 轮询。
- **如何复现结果？** 传入 `seed` 参数，相同 seed + 相同参数会生成相似结果。
- **img2video 模式下宽高比如何处理？** 默认 `adaptive` 会根据输入图片自动选择最佳宽高比，也可以手动指定，系统会自动裁剪。
- **fps 能否调整？** Seedance Pro Fast 固定 24fps，不可调整。
- **固定镜头有什么用？** 启用 `camera_fixed` 后，镜头位置保持不变，适合需要稳定画面的场景。
