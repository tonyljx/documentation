## Submit task

- Send `Authorization: Bearer <your_api_key>`.
- Required: `prompt`, `mode`; `img2video` requires `input_image_url`.
- Optional: `duration`, `resolution`, `aspect_ratio`, `seed`, `watermark`, `camera_fixed`.
- Async `task_id`; poll `/api/v1/task/status`.

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-pro-fast \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A cat playing piano in a cozy living room",
    "mode": "text2video",
    "duration": 5,
    "resolution": "480p",
    "aspect_ratio": "16:9"
  }'
```

<Note>
- `text2video`: no `input_image_url`.  
- `img2video`: `input_image_url` required; images jpeg/png/webp/bmp/tiff/gif, aspect 0.4-2.5, short side >300px, long side &lt;6000px, size &lt;30MB.  
- Fixed 24fps.
</Note>
