## 提交任务

- 传 `Authorization: Bearer <your_api_key>`。
- 必填 `prompt`、`mode` (`std`/`pro`)、`duration` (`5`/`10`)；`image` 与 `image_tail` 至少提供一个。
- 可选：`model_name`、`negative_prompt`、`cfg_scale`、`static_mask`、`dynamic_masks`、`camera_control`、`callback_url`、`external_task_id`。
- `image_tail` 与 `dynamic_masks`/`static_mask`/`camera_control` 互斥；图片需 jpg/jpeg/png，≤10MB，宽高 ≥300px，宽高比 1:2.5~2.5:1。
- 异步返回 `task_id`，使用 `/api/v1/task/status` 轮询。

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/kling-ai \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "主角在森林中漫步",
    "mode": "pro",
    "duration": 10,
    "image": "https://example.com/forest.jpg",
    "model_name": "kling-v2-master",
    "camera_control": {
      "type": "forward_up"
    }
  }'
```

<Note>
- 动态笔刷：`dynamic_masks` 最多 6 组，5s 轨迹长度 ≤77 点，坐标原点为左下角。
- 镜头控制：`camera_control.type = simple` 时 `config` 六项仅能有 1 个非零（范围 -10~10）。
</Note>
