## Submit task

- Send `Authorization: Bearer <your_api_key>`.
- Required: `prompt`, `mode` (`std`/`pro`), `duration` (`5`/`10`), and at least one of `image` or `image_tail`.
- Optional: `model_name`, `negative_prompt`, `cfg_scale`, `static_mask`, `dynamic_masks`, `camera_control`, `callback_url`, `external_task_id`.
- `image_tail` is mutually exclusive with `dynamic_masks` / `static_mask` / `camera_control`. Images must be jpg/jpeg/png, ≤10MB, min 300px side, aspect 1:2.5–2.5:1.
- Async `task_id`; poll `/api/v1/task/status`.

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/kling-ai \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "The main character walks through a forest",
    "mode": "pro",
    "duration": 10,
    "image": "https://example.com/forest.jpg",
    "model_name": "kling-v2-master",
    "camera_control": {
      "type": "forward_up"
    }
  }'
```

<Note>
- `dynamic_masks`: up to 6 sets; 5s trajectories ≤77 points; origin bottom-left.
- `camera_control.type = simple`: only one non-zero config param (`horizontal`/`vertical`/`pan`/`tilt`/`roll`/`zoom`, range -10~10).
</Note>
