## Submit task

- Header: `Authorization: Bearer <your_api_key>`.
- Required: `prompt`. Optional: `image_urls` (max 8, 30MB each), `aspect_ratio`, `resolution`, `output_format`.
- Async `task_id` and `credits_deducted`; poll `/api/v1/task/status`.
- **Breaking:** `image_input` deprecated; use `image_urls`.

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/nano-banana-pro \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A beautiful sunset over the ocean",
    "image_urls": [
      "https://example.com/reference1.jpg",
      "https://example.com/reference2.jpg"
    ],
    "aspect_ratio": "16:9",
    "resolution": "2K",
    "output_format": "png"
  }'
```

<Note>
- Aspect ratios: `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `4:5`, `5:4`, `9:16`, `16:9`, `21:9`, `auto` (default `1:1`).  
- Credits: `1K/2K = 10`, `4K = 20`.  
- Output formats: `png` / `jpg` (default `png`).
</Note>
