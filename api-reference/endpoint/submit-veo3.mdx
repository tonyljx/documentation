## Submit task

- Send `Authorization: Bearer <your_api_key>`.
- Required: `prompt`. Optional: `image_urls`, `generation_type`, `model`, `aspect_ratio`, `seeds`, `enable_translation`, `watermark`.
- Returns `task_id`; poll `/api/v1/task/status`.

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A dog running through a neon city at night",
    "model": "veo3_fast",
    "aspect_ratio": "16:9"
  }'
```

<Note>
- `TEXT_2_VIDEO` must not include `image_urls`; other modes allow up to 2 refs (`REFERENCE_2_VIDEO` up to 3, 16:9, `veo3_fast`).
- `callBackUrl` is deprecated; use polling.
</Note>
