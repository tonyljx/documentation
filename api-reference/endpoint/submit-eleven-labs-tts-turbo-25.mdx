## Submit task

- Header: `Authorization: Bearer <your_api_key>`.
- Required: `text` (1-5000 characters).
- Optional: `voice`, `stability`, `similarity_boost`, `style`, `speed`, `timestamps`, `previous_text`, `next_text`, `language_code`.
- Async `task_id`; poll `/api/v1/task/status`.
- **50% cheaper** than TTS V2, optimized for low-latency.

### Basic example

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/eleven-labs-tts-turbo-25 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello, welcome to our AI platform."
  }'
```

### Full parameters example

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/eleven-labs-tts-turbo-25 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Unlock powerful API with our platform.",
    "voice": "Brian",
    "stability": 0.5,
    "speed": 1.0
  }'
```

<Note>
- **Text**: 1-5000 characters, multi-language supported.
- **Voice**: 21 options (Rachel, Brian, Sarah, etc.). Default: Rachel.
- **Speed**: 0.7-1.2 (default: 1.0).
- **Pricing**: 40 credits per 1000 characters (50% cheaper than V2).
</Note>
