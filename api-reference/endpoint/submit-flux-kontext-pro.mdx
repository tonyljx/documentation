## Submit task

- Send `Authorization: Bearer <your_api_key>`.
- Required: `prompt`. Optional: `aspect_ratio`, `image_urls` (max 4), `output_format`, `prompt_upsampling`, `safety_tolerance`, `seed`, `webhook_url`, `webhook_secret`.
- Async `task_id`; poll `/api/v1/task/status`.
- **Breaking:** `input_image`/`input_image_2`/`_3`/`_4` deprecated; use `image_urls` array.

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/flux-kontext-pro \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A futuristic city at night with neon lights",
    "aspect_ratio": "16:9",
    "image_urls": [
      "https://example.com/reference1.jpg"
    ],
    "output_format": "png",
    "safety_tolerance": 2
  }'
```

<Note>
- Cost: base 15-20 credits depending on aspect ratio and reference count.  
- Aspect ratios 21:9 to 9:21 (`width:height`, e.g., "16:9", "1:1", "9:16").  
- References: up to 4 (experimental).  
- Output: `jpeg`/`png` (default `png`).  
- Safety: 0-6 (0 strict, 6 lax), default 2.
</Note>
