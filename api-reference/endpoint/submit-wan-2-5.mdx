## Submit task

- Header: `Authorization: Bearer <your_api_key>`.
- Required: `prompt` (max 800 chars), `type` (`text-to-video` or `image-to-video`).
- Optional: `duration` (5/10s), `resolution` (720p/1080p), `aspect_ratio` (text-to-video only), `image_urls`, `negative_prompt`, `enable_prompt_expansion`, `seed`.
- Async `task_id` and `credits_deducted`; poll `/api/v1/task/status`.

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/wan-2-5 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A majestic dragon flying through clouds at sunset",
    "type": "text-to-video",
    "duration": 10,
    "resolution": "1080p",
    "aspect_ratio": "16:9"
  }'
```

### Image-to-video example

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/wan-2-5 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "The character turns around and smiles",
    "type": "image-to-video",
    "image_urls": ["https://example.com/character.jpg"],
    "duration": 5,
    "resolution": "720p"
  }'
```

<Note>
- **Type**: `text-to-video` (no images) or `image-to-video` (1 image required).
- **Duration**: `5` or `10` seconds (default: 5).
- **Resolution**: `720p` or `1080p` (default: 1080p).
- **Aspect ratio**: `16:9`, `9:16`, `1:1` (text-to-video only, default: 16:9).
- **Pricing**: 720p = 60 credits/s, 1080p = 100 credits/s.
- **Prompt**: Max 800 characters; negative prompt max 500 characters.
</Note>
