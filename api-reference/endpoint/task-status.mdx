## Query task

- Send `task_id`; include `Authorization: Bearer <your_api_key>`.
- Returns `status` (`PROCESSING`/`SUCCEEDED`/`FAILED`); success includes `result_urls`.

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/status \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "task_id": "local-task-id"
  }'
```
