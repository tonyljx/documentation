## Quick start

- Platform: Wizzx gateway; use `Authorization: Bearer <your_api_key>`.
- Submit job: `POST https://api.wizzx.ai/api/v1/task/submit/veo3` (async `task_id`).
- Poll status: `POST https://api.wizzx.ai/api/v1/task/status` with `task_id`.
- Default model `veo3_fast`, aspect ratio 16:9; supports text-only or 1-3 reference images.

### Auth

Every request:

```bash
Authorization: Bearer <your_api_key>
```

Wizzx gateway validates and forwards; no extra params needed.

### Submit job

Endpoint: `POST https://api.wizzx.ai/api/v1/task/submit/veo3`, `Content-Type: application/json`

| Field | Type | Description | Default/Rules |
| --- | --- | --- | --- |
| `prompt` | string | Video description | **Required** |
| `image_urls` | `array<string>` | Reference images (1-3) | Omit for text-only; 1-2 = first/last frames; 3 requires `REFERENCE_2_VIDEO` |
| `generation_type` | string | `TEXT_2_VIDEO` / `FIRST_AND_LAST_FRAMES_2_VIDEO` / `REFERENCE_2_VIDEO` | Inferred from `image_urls` if missing |
| `model` | string | `veo3` / `veo3_fast` | Default `veo3_fast`; `REFERENCE_2_VIDEO` only `veo3_fast` |
| `aspect_ratio` | string | `16:9` / `9:16` / `Auto` | Default `16:9`; `REFERENCE_2_VIDEO` only 16:9 |
| `seeds` | int | 10000-99999 | Optional |
| `enable_translation` | bool | Auto translate prompt | Default `true` |
| `watermark` | string | Custom watermark | Optional |

<Warning>
  `callBackUrl` is deprecated; use polling. `TEXT_2_VIDEO` rejects `image_urls`; other modes max 2 images ( `REFERENCE_2_VIDEO` up to 3, must be 16:9 and `veo3_fast`).
</Warning>

#### Request: Text2Video

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A dog running through a neon city at night",
    "aspect_ratio": "16:9",
    "model": "veo3_fast",
    "seeds": 12345
  }'
```

#### Request: Image2Video (first/last frames)

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Make the scene dynamic while keeping the character consistent",
    "image_urls": [
      "https://example.com/frame1.jpg",
      "https://example.com/frame2.jpg"
    ],
    "generation_type": "FIRST_AND_LAST_FRAMES_2_VIDEO",
    "aspect_ratio": "9:16"
  }'
```

#### Request: Reference2Video

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Product turntable style, soft studio light",
    "image_urls": [
      "https://example.com/ref1.jpg",
      "https://example.com/ref2.jpg",
      "https://example.com/ref3.jpg"
    ],
    "generation_type": "REFERENCE_2_VIDEO",
    "model": "veo3_fast",
    "aspect_ratio": "16:9",
    "enable_translation": true
  }'
```

Success response:

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "provider_task_id": "veo_task_abcdef123456",
    "model_key": "veo3",
    "provider_key": "veo3"
  }
}
```

### Job status

Endpoint: `POST https://api.wizzx.ai/api/v1/task/status`

Body:

```json
{
  "task_id": "local-task-id"
}
```

Response:

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.example.com/video.mp4"
    ],
    "error": ""
  }
}
```

Status: `PROCESSING`, `SUCCEEDED` (use `result_urls`), `FAILED` (see `error`).

### Params at a glance

- Required: `prompt`
- Optional: `image_urls`, `generation_type`, `model`, `aspect_ratio`, `seeds`, `enable_translation`, `watermark`
- Rules:
  - `REFERENCE_2_VIDEO`: `image_urls` 1-3, `aspect_ratio` 16:9, `model` veo3_fast
  - `TEXT_2_VIDEO`: no `image_urls`
  - Others: `image_urls` max 2

### FAQ

- Callback required? No, poll `task/status`.
- Reproducibility? Provide `seeds` (10000-99999).
- Non-English prompts? Auto-translated when `enable_translation=true` (default).
