## 快速上手

- 品牌：Wizzx（第三方 API 聚合），统一使用 `Authorization: Bearer <your_api_key>`.
- 提交生成：`POST /api/v1/task/submit/veo3`，异步返回 `task_id`。
- 轮询状态：`POST /api/v1/task/status`，传入 `task_id` 获取结果或错误。
- 默认模型 `veo3_fast`，16:9 画幅；支持文本驱动和 1-3 张图像驱动。

### 认证

所有请求需携带：

```bash
Authorization: Bearer <your_api_key>
```

Wizzx 网关会校验并注入身份，无需额外参数。

### 提交任务

接口：`POST /api/v1/task/submit/veo3`，`Content-Type: application/json`

字段 | 类型 | 说明 | 默认/规则
-|-|-|-
`prompt` | string | 视频描述 | **必填**
`image_urls` | array<string> | 参考图，1-3 张 | 不传则纯文本；1-2 张默认首尾帧；3 张需配合 `REFERENCE_2_VIDEO`
`generation_type` | string | `TEXT_2_VIDEO` / `FIRST_AND_LAST_FRAMES_2_VIDEO` / `REFERENCE_2_VIDEO` | 未指定时根据 `image_urls` 推断
`model` | string | `veo3` / `veo3_fast` | 默认 `veo3_fast`；`REFERENCE_2_VIDEO` 仅支持 `veo3_fast`
`aspect_ratio` | string | `16:9` / `9:16` / `Auto` | 默认 `16:9`；`REFERENCE_2_VIDEO` 仅 16:9
`seeds` | int | 10000-99999，可复现 | 可选
`enable_translation` | bool | 提示词自动英译 | 默认 `true`
`watermark` | string | 自定义水印 | 可选

<Callout type="warning">
  `callBackUrl` 已弃用；使用轮询获取结果。`TEXT_2_VIDEO` 不接受 `image_urls`，其他模式最多 2 张图（`REFERENCE_2_VIDEO` 最多 3 张且须 16:9、`veo3_fast`）。
</Callout>

#### 请求示例（Text2Video）

```bash
curl -X POST https://api.example.com/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A dog running through a neon city at night",
    "aspect_ratio": "16:9",
    "model": "veo3_fast",
    "seeds": 12345
  }'
```

#### 请求示例（Image2Video，首尾帧）

```bash
curl -X POST https://api.example.com/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Make the scene dynamic while keeping the character consistent",
    "image_urls": [
      "https://example.com/frame1.jpg",
      "https://example.com/frame2.jpg"
    ],
    "generation_type": "FIRST_AND_LAST_FRAMES_2_VIDEO",
    "aspect_ratio": "9:16"
  }'
```

#### 请求示例（Reference2Video）

```bash
curl -X POST https://api.example.com/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Product turntable style, soft studio light",
    "image_urls": [
      "https://example.com/ref1.jpg",
      "https://example.com/ref2.jpg",
      "https://example.com/ref3.jpg"
    ],
    "generation_type": "REFERENCE_2_VIDEO",
    "model": "veo3_fast",
    "aspect_ratio": "16:9",
    "enable_translation": true
  }'
```

成功响应示例：

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "provider_task_id": "veo_task_abcdef123456",
    "model_key": "veo3",
    "provider_key": "veo3"
  }
}
```

### 任务查询

接口：`POST /api/v1/task/status`

请求体：

```json
{
  "task_id": "local-task-id"
}
```

响应示例：

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.example.com/video.mp4"
    ],
    "error": ""
  }
}
```

状态值：`PROCESSING`（生成中）、`SUCCEEDED`（可用 `result_urls`）、`FAILED`（查看 `error`）。

### 参数速览与规则

- 必填：`prompt`
- 可选：`image_urls`、`generation_type`、`model`、`aspect_ratio`、`seeds`、`enable_translation`、`watermark`
- 规则提示：
  - `REFERENCE_2_VIDEO`：`image_urls` 1-3 张，`aspect_ratio`=16:9，`model`=veo3_fast
  - `TEXT_2_VIDEO`：不传 `image_urls`
  - 其他模式：`image_urls` 最多 2 张

### 计费（占位）

当前固定 10 积分/次，后续变更以 Wizzx 公告为准。

### 常见问题

- 需要回调吗？不需要，使用 `task/status` 轮询。
- 如何复现结果？传入 `seeds`（10000-99999）。
- 非英文提示词会怎样？默认 `enable_translation=true` 自动翻译，可关闭。
