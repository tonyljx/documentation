## Quick start

- **Breaking change (2025-12-10)**: `generation_type` → `type`; enum values now `text-to-video` / `image-to-video` / `reference-to-video`.
- Platform: Wizzx gateway; Auth header `Authorization: Bearer <your_api_key>`.
- Submit: `POST https://api.wizzx.ai/api/v1/task/submit/veo3` → returns `task_id`.
- Poll: `POST https://api.wizzx.ai/api/v1/task/status` with `task_id`.
- Default model `veo3_fast`, aspect ratio `16:9`; supports text-only, first/last frames, or reference images.

### Auth

Every request:

```bash
Authorization: Bearer <your_api_key>
```

Wizzx gateway validates and forwards; no extra params needed.

### Submit job

Endpoint: `POST https://api.wizzx.ai/api/v1/task/submit/veo3`, `Content-Type: application/json`

### Required

| Field | Type | Notes |
| --- | --- | --- |
| `prompt` | string | Video description |

### Optional

| Field | Type | Default | Notes |
| --- | --- | --- | --- |
| `type` | string | Auto | `text-to-video` / `image-to-video` / `reference-to-video` |
| `image_urls` | []string | - | 1-3 images (see rules below) |
| `model` | string | `veo3_fast` | `veo3` / `veo3_fast` |
| `aspect_ratio` | string | `16:9` | `16:9` / `9:16` / `Auto` |
| `seeds` | int | - | 10000-99999 |
| `enable_translation` | bool | true | Auto-translate prompt to English |
| `watermark` | string | - | Custom watermark text |

### Type rules

| type | Description | `image_urls` requirement |
|------|-------------|--------------------------|
| `text-to-video` | Pure text to video | Omit `image_urls` |
| `image-to-video` | First frame / first+last | 1-2 images (1 = first frame, 2 = first+last) |
| `reference-to-video` | Reference images | 1-3 images, must be `16:9` + `model=veo3_fast` |

Auto inference:
- No `image_urls`: default `text-to-video`
- 1-2 `image_urls`: default `image-to-video`
- 1-3 `image_urls` + explicit `type="reference-to-video"`: reference mode

<Warning>
  `reference-to-video` requires `veo3_fast` and `aspect_ratio`=`16:9`.
</Warning>

#### Request: Text-to-Video

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A dog running through a neon city at night",
    "type": "text-to-video",
    "aspect_ratio": "16:9",
    "model": "veo3_fast",
    "seeds": 12345
  }'
```

#### Request: Image-to-Video (first/last frames)

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Make the scene dynamic while keeping the character consistent",
    "type": "image-to-video",
    "image_urls": [
      "https://example.com/frame1.jpg",
      "https://example.com/frame2.jpg"
    ],
    "aspect_ratio": "9:16"
  }'
```

#### Request: Reference-to-Video (16:9 + veo3_fast)

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/veo3 \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Product turntable style, soft studio light",
    "type": "reference-to-video",
    "image_urls": [
      "https://example.com/ref1.jpg",
      "https://example.com/ref2.jpg",
      "https://example.com/ref3.jpg"
    ],
    "model": "veo3_fast",
    "aspect_ratio": "16:9",
    "enable_translation": true
  }'
```

Success response:

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "provider_task_id": "veo_task_abcdef123456",
    "model_key": "veo3",
    "provider_key": "veo3"
  }
}
```

### Job status

Endpoint: `POST https://api.wizzx.ai/api/v1/task/status`

Body:

```json
{
  "task_id": "local-task-id"
}
```

Response:

```json
{
  "code": 0,
  "data": {
    "task_id": "local-task-id",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.example.com/video.mp4"
    ],
    "error": ""
  }
}
```

Status: `PROCESSING`, `SUCCEEDED` (use `result_urls`), `FAILED` (see `error`).

### FAQ

- Where did `generation_type` go? Unified to `type`, enum now lower-case with hyphens.
- Need callback URL? No, poll `task/status`.
- How to reproduce results? Provide `seeds` (10000-99999).
- Non-English prompt? Auto-translated when `enable_translation=true` (default).
- Difference between 1 vs 2 images in image-to-video? One image = first frame; two images = first + last frame with auto transition.
