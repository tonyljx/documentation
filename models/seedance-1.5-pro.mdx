## Quick start

- Platform: Wizzx gateway; `Authorization: Bearer <your_api_key>`.
- Submit: `POST https://api.wizzx.ai/api/v1/task/submit/seedance-1.5-pro` â†’ returns `task_id`.
- Poll: `POST https://api.wizzx.ai/api/v1/task/status` with `task_id`.
- Modes: `text-to-video` / `image-to-video`; fixed 24fps; duration 4-12s; resolutions 480p/720p.
- **Key feature**: `generate_audio` - generates video with synchronized audio (voice, sound effects, background music).

### Auth

```bash
Authorization: Bearer <your_api_key>
```

### Submit job

Endpoint: `POST https://api.wizzx.ai/api/v1/task/submit/seedance-1.5-pro`, `Content-Type: application/json`

### Required

| Field | Type | Notes |
| --- | --- | --- |
| `prompt` | string | Video description |
| `type` | string | `text-to-video` or `image-to-video` |

### Optional

| Field | Type | Default | Notes |
| --- | --- | --- | --- |
| `image_urls` | []string | - | Required for `image-to-video`, max 2 images (first frame + last frame) |
| `duration` | int | 5 | `4` to `12` seconds, or `-1` for auto |
| `resolution` | string | 720p | `480p` / `720p` (no 1080p support) |
| `aspect_ratio` | string | adaptive | `16:9`, `4:3`, `1:1`, `3:4`, `9:16`, `21:9`, `adaptive` |
| `seed` | int | -1 | RNG seed; -1 for random, range [0, 4294967295] |
| `watermark` | bool | false | Add watermark |
| `camera_fixed` | bool | false | Lock camera position |
| `generate_audio` | bool | true | Generate synchronized audio (voice, sound effects, music) |
| `return_last_frame` | bool | false | Return the last frame image (useful for consecutive video generation) |

<Warning>
  **Image requirements (image-to-video)**
  - Formats: jpeg/png/webp/bmp/tiff/gif
  - Aspect ratio: 0.4-2.5
  - Short side \>300px, long side \<6000px
  - Size \<30MB
  - Max 2 images: first image = first frame, second image = last frame
</Warning>

<Note>
  **First + Last Frame Support**
  Seedance 1.5 Pro supports first frame + last frame image-to-video generation. Provide 2 images in `image_urls` to control both the start and end of your video.
</Note>

#### Example: Text2Video with Audio

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-1.5-pro \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A cat playing piano in a cozy living room, gentle jazz music playing",
    "type": "text-to-video",
    "duration": 8,
    "resolution": "720p",
    "aspect_ratio": "16:9",
    "generate_audio": true
  }'
```

#### Example: Text2Video without Audio

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-1.5-pro \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A serene mountain landscape with flowing clouds",
    "type": "text-to-video",
    "duration": 5,
    "resolution": "480p",
    "generate_audio": false
  }'
```

#### Example: Image2Video (First + Last Frame)

```bash
curl -X POST https://api.wizzx.ai/api/v1/task/submit/seedance-1.5-pro \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "The character walks forward gracefully and waves at the camera",
    "type": "image-to-video",
    "image_urls": [
      "https://example.com/first_frame.jpg",
      "https://example.com/last_frame.jpg"
    ],
    "duration": 6,
    "resolution": "720p",
    "generate_audio": true
  }'
```

Success response:

```json
{
  "code": 0,
  "data": {
    "task_id": "task_123456789",
    "model_key": "seedance-1.5-pro"
  }
}
```

### Query status

Endpoint: `POST https://api.wizzx.ai/api/v1/task/status`

```json
{
  "task_id": "task_123456789"
}
```

Response example:

```json
{
  "code": 0,
  "data": {
    "task_id": "task_123456789",
    "status": "SUCCEEDED",
    "result_urls": [
      "https://cdn.example.com/video.mp4"
    ],
    "error": ""
  }
}
```

Status values: `PROCESSING`, `SUCCEEDED`, `FAILED`.

### Seedance 1.5 Pro vs Seedance Pro Fast

| Feature | Seedance 1.5 Pro | Seedance Pro Fast |
|---------|------------------|-------------------|
| Audio generation | Yes (default on) | No |
| First + Last frame | Yes (2 images) | No (1 image only) |
| Resolutions | 480p, 720p | 480p, 720p, 1080p |
| Duration range | 4-12s or auto | 5s or 10s |
| Auto duration | Yes (-1) | No |

### FAQ

- **What's unique about Seedance 1.5 Pro?** Audio generation and first+last frame support for more controlled video creation.
- **How does `generate_audio` work?** When enabled (default), the model generates synchronized audio including voice, sound effects, and background music based on your prompt.
- **What's the difference between 1.5 Pro and Pro Fast?** 1.5 Pro adds audio generation and first+last frame control, but doesn't support 1080p resolution. Pro Fast is silent-only but supports higher resolution.
- **How to create consecutive videos?** Use `return_last_frame: true` to get the last frame, then use it as the first frame for the next video.
- **Why no 1080p?** Seedance 1.5 Pro currently only supports 480p and 720p resolutions.
- **Can fps be changed?** Fixed at 24fps.
